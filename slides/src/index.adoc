== Un cahier des charges bien rempli

== Définition textuelle simple

Le langage: AML, Automaton markup language

Simple = intuitif et compact

== Exemple de langage

....
finite-automaton {
    states {
        0[initial] 1 2[terminal]
    }
    symbols {
        a b
    }
    transitions {
        0 b -> 2
        1 a -> 2
        1 b -> 0
    }
}
....
